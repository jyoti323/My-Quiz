# My-Quiz
My first quiz project using JavaScript! It takes user answers, checks correctness, and shows the final score.

## Learning Objectives

-  Use arrays to store questions and answers
-  Create functions to organize code
-  Use for loops to repeat actions
-  Get user input with prompt()
-  Process text with toLowerCase() and trim()
-  Check answers with if statements
-  Track score with variables

  ## Technologies Used

- JavaScript (ES6)
- html
